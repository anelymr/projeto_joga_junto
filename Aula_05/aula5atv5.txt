1. O que são estruturas de repetição e iteração?

    As estruturas de repetição (também conhecidas como loops) são recursos em programação que:
    - Permitem executar um bloco de código várias vezes, com base em uma condição ou um conjunto de regras
    - São usadas para automatizar tarefas que exigem repetição, como processamento de listas de dados, contagem ou execução de ações até que uma determinada condição seja satisfeita.

    A iteração é o ato de repetir um processo em um programa, geralmente usando uma estrutura de repetição, como um loop, para acessar e processar cada elemento de um conjunto de dados ou para executar uma ação várias vezes.

2. Quando são usadas?
    Quando você deseja realizar a mesma operação várias vezes, sem repetir o código manualmente.
    Elas são úteis para percorrer listas de elementos, realizar cálculos iterativos, ler ou escrever em arquivos, validar entradas do usuário e implementar algoritmos que envolvem repetição controlada.

3. Quais os principais tipos de estruturas de repetição?
    For(para): É usado quando você sabe antecipadamente quantas vezes deseja repetir um bloco de código.
    Ele geralmente é usado para percorrer uma sequência, como uma lista ou uma faixa de números, e executa um conjunto de instruções para cada elemento da sequência.
    Em Python, um exemplo de loop "for" seria:
       "for i in range(5):
            print(i)"

    While(enquanto): É usado quando você deseja repetir um bloco de código enquanto uma condição específica for verdadeira.
    Ele pode ser usado para situações em que você não sabe antecipadamente quantas vezes o loop será executado.
    Um exemplo de loop "while" em Python seria:
        "contador = 0
         while contador < 5:
            print(contador)
            contador += 1"
